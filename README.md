# Raccourcis clavier
---
Retrouvez ici les raccourcis claviers qui vous seront utiles pour votre parcours à la toile. L'utilisation de raccourcis clavier peut grandement améliorer votre efficacité et votre confort de travail. Pas la peine de chercher à les mémoriser tous d'un coup.
Si vous avez l'impression de retrouver les mêmes actions dans votre travail quotidien, cherchez s'il existe un raccourci clavier pour ces actions, et essayez de les utiliser à la place de la souris.
N'hésitez pas à les pratiquer régulièrement pour les mémoriser, et à les partager avec vos camarades quand l'occasion se présente.

## Raccourcis généraux
- `Ctrl`  + `S` : Sauvegarder votre travail
- `Ctrl` + `Z` : Annuler la dernière action
- `Ctrl` + `Y` : Rétablir la dernière action annulée
- `Ctrl` + `C` : Copier
- `Ctrl` + `V` : Coller
- `Ctrl` + `X` : Couper
- `Ctrl` + `A` : Sélectionner tout
- `Ctrl` + `F` : Rechercher
- `Ctrl` + `N` : Ouvrir une nouvelle fenêtre ou un nouvel onglet (selon l'application)
- `Fn` + `←` : Aller au début de la ligne
- `Fn` + `→` : Aller à la fin de la ligne

## Raccourcis windows
- `Win` : Ouvrir le menu Démarrer (puis commencez à écrire pour rechercher une application, un fichier ou une fonctionnalité)
- Gestion des fenêtres (par exemple, pour afficher votre éditeur de code et votre navigateur côte à côte) :
  - `Win` + `←` : Ancrer la fenêtre active à gauche de l'écran
  - `Win` + `→` : Ancrer la fenêtre active à droite de l'écran
  - `Win` + `↑` : Agrandir la fenêtre active
  - `Win` + `↓` : Réduire la fenêtre active
- `Alt` + `Tab` : Basculer entre les applications ouvertes. (Maintenez `Alt` et appuyez sur `Tab` pour parcourir les applications, puis relâchez `Alt` pour ouvrir l'application souhaitée)
- `Win` + `D` : Afficher le bureau (minimiser toutes les fenêtres)
- `Win` + `V` : Ouvrir le presse-papiers (affiche l'historique des éléments copiés, vous permettant de coller un élément précédent). Note : Cette fonctionnalité doit être activée dans les paramètres de Windows pour fonctionner.
[Activer l'historique du presse-papiers sur Windows](https://www.microsoft.com/fr-fr/windows/tips/clipboard-history#:~:text=Appuyez%20sur%20la%20touche%20Windows,sur%20la%20touche%20Windows%20%2B%20V.)

## Navigateur web
- `Ctrl` + `T` : Ouvrir un nouvel onglet
- `Ctrl` + `W` : Fermer l'onglet actuel
- `Ctrl` + `Shift` + `T` : Rouvrir le dernier onglet fermé
- `Ctrl` + `Tab` : Passer à l'onglet suivant
- `Ctrl` + `Shift` + `Tab` : Passer à l'onglet précédent
- `Ctrl` + `L` : Placer le curseur dans la barre d'adresse (sélectionne l'URL actuelle). Vous pouvez enchainer avec `Ctrl` + `C` pour copier l'URL, ou commencer à taper pour rechercher ou entrer une nouvelle URL.
- `Fn` + `←` : Aller en haut de la page
- `Fn` + `→` : Aller en bas de la page

## Visual Studio Code
### Gérer l'interface, les fenêtres
- `Ctrl` + `B` : Afficher ou masquer la barre latérale
- `Ctrl` + `Shift` + `E` : Afficher l'explorateur de fichiers
- `Ctrl` + `Tab` : Naviguer dans les derniers fichiers modifiés (maintenez `Ctrl` et appuyez sur `Tab` pour parcourir les fichiers, puis relâchez `Ctrl` pour ouvrir le fichier souhaité)
- `Ctrl` + `Shift` + `Tab` : Naviguer dans les derniers fichiers modifiés dans l'ordre inverse
- `Ctrl` + `P` : Ouvrir rapidement un fichier (tapez le nom du fichier pour le trouver)
- `Ctrl` + `Shift` + `P` : Ouvrir la palette de commandes (tapez une commande pour l'exécuter)

### Travailler dans un fichier
- Déplacement du curseur :
  - `Ctrl` + `Flèche gauche/droite` : Se déplacer d'un mot à la fois
  - `Fn` + `←` : Aller au début de la ligne
  - `Fn` + `→` : Aller à la fin de la ligne
  - `Ctrl` + `Entrée` : Nouvelle ligne en dessous de la ligne actuelle, sans modifier la ligne actuelle (utile pour ajouter du code entre deux lignes déjà écrites, ou ajouter du contenu au sein d'une balise HTML)
  - `Alt` + `↓` : Déplacer la ligne de code actuelle vers le bas
  - `Alt` + `↑` : Déplacer la ligne de code actuelle vers le haut
  - `Alt` + `Shift` + `↓` : Dupliquer la ligne de code actuelle vers le bas
  - `Alt` + `Shift` + `↑` : Dupliquer la ligne de code actuelle vers le haut
  - `Tab`, _lorsque le curseur est en début de ligne, ou lorsque du contenue est sélectionné_ : Décaler le code vers la droite (augmente le niveau d'indentation) pour la ligne actuelle ou les lignes sélectionnées
  - `Shift` + `Tab` : Décaler le code vers la gauche (diminue le niveau d'indentation)
- Formatage
  - `Alt` + `Shift` + `F` : Formater le code (**indentation automatique**)

- Modifier l'affichage :
  - `Alt` + `Z` : Activer ou désactiver le **retour à la ligne automatique** (utile pour les fichiers avec de longues lignes de code ou de texte)
  - Gérer le **zoom** :
    - `Ctrl` + `+` : Zoom avant
    - `Ctrl` + `-` (sur le pavé numérique): Zoom arrière.
    ⚠️ Sur certains claviers, le `-` du pavé numérique peut ne pas fonctionner pour le zoom arrière. Dans ce cas :
    - Zoom arrière (alternative) : ouvrir la palette de commandes (`Ctrl` + `Shift` + `P`), puis taper "Zoom" et sélectionner "Zoom arrière".
- Modifier le contenu :
  - `Ctrl` + `/` : **Commenter** ou décommenter la ligne de code actuelle
  - `Ctrl` + `Shift` + `K` : Supprimer la ligne de code actuelle

- **Rechercher, sélectionner** :
  - `Ctrl` + `F` : Rechercher dans le fichier actuel
  - `Ctrl` + `Shift` + `F` : Rechercher dans tous les fichiers
  - _Double-clic sur un mot_ : Sélectionner le mot entier. Utile pour sélectionner une variable ou une fonction rapidement.
  - _Triple-clic sur une ligne_ : Sélectionner la ligne entière. Utile pour sélectionner rapidement une ligne de code.
  - `Ctrl` + `D` : Sélectionner le mot actuel, puis les occurrences suivantes du même mot (appuyez plusieurs fois pour sélectionner toutes les occurrences). Utile pour renommer une variable ou une fonction dans tout le fichier.
